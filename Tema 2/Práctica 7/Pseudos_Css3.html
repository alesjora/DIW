:link

Representa un elemento que aún no se ha visitado. Coincide con cada elemento no visitado <a>, <area>, o <link> que tiene un atributo href.

Sintaxis: :link

#ejemploLink{
background-color: green; }
#ejemploLink:link {
background-color: yellow; }
Visita la página y cambiará el color de fondo.

:visited

Representa enlaces que el usuario ya ha visitado. Para darle un estilo apropiado a los enlaces, coloque la regla :visited después de la regla :link pero antes de las reglas :hover y :active, según lo definido por el orden LVHA: :link — :visited — :hover — :active.

Sintaxis: :visited

#ejemploVisited {
background-color: green; }
#ejemploVisited:visited {
background-color: yellow; }
Visita la página y cambiará el color de fondo.

:hover

Coincide cuando el usuario interactúa con un elemento con un dispositivo señalador, pero no necesariamente lo activa. Generalmente se activa cuando el usuario se desplaza sobre un elemento con el cursor (puntero del mouse).

Sintaxis: :hover

#ejemploHover:hover {
background-color: blueviolet; }
Pasa el ratón por encima para cambiar el color.

:active

Representa un elemento (como un botón) que el usuario está activando. Cuando se usa un mouse, la "activación" generalmente comienza cuando el usuario presiona el botón primario del mouse y termina cuando se suelta.

Sintaxis: :active

#ejemploActive:active {
background-color: gray;
color: white; }
Mientras tengas pulsado el enlace el color de fondo será gris y la letra blanca.

:focus

Representa un elemento (como una entrada de formulario) que ha recibido el foco. Generalmente se activa cuando el usuario hace clic, toca un elemento o lo selecciona con la tecla "Tab" del teclado.

Sintaxis: :focus

#ejemploFocus:focus {
background-color: blue;
} }

:enabled

Representa cualquier elemento habilitado. Un elemento está habilitado si puede ser activado o si acepta el foco.

Sintaxis: :enabled

Los elementos habilitados se mostrarán en verde



:disabled

Representa a cualquier elemento deshabilitado. Un elemento se encuentra deshabilitado si no puede ser activado ni aceptar el foco.

Sintaxis: :disabled

Los elementos deshabilitado se mostrarán en rojo



:checked

Representa cualquier radio, checkbox u option que está marcado o conmutado a un estado on.

Sintaxis: :checked

Los elementos que estén marcados tendrán una sombra azul

:indeterminate

Representa cualquier elemento de formulario cuyo estado sea indeterminado. Es decir, un elemento que no está "inicializado" y no tiene ningún valor

Sintaxis: :indeterminate

Los elementos indeterminados tendrán un fondo amarillo. Al pulsar en alguno el fondo se quitará ya que el elemento obtendrá un valor

:root

Selecciona el elemento raíz de un árbol que representa el documento. En HTML, :root representa el elemento html y es idéntico al selector html, excepto que su especificidad es mayor.

Sintaxis: :root

El código siguiente obtendría el elemento html cambiaría su fondo a azul

:root { background: blue; }

:nth-child()

Coincide con uno o más elementos en función de su posición entre un grupo de hermanos. Se especifica con un único argumento, que representa el patrón para los elementos coincidentes. Algunos valores son odd que representa elementos cuya posición numérica en una serie de hermanos es impar: 1, 3, 5, etc. Y even que representa elementos cuya posición numérica en una serie de hermanos es par: 2, 4, 6, etc.

Sintaxis: :nth-child(an-plus-b | even | odd)

En el siguiente ejemplo usaremos nth-child(odd) que coloreará los elementos impares de color azul

:nth-last-child()

Selecciona uno o más elementos en función de su posición entre un grupo de hermanos, contando desde el final. Esta pseudo-clase es esencialmente la misma que :nth-child, excepto que cuenta los elementos hacia atrás desde el final, no hacia adelante desde el principio. Algunos valores son odd que representa elementos cuya posición numérica en una serie de hermanos es impar: 1, 3, 5, etc. Y even que representa elementos cuya posición numérica en una serie de hermanos es par: 2, 4, 6, etc.

Sintaxis: :nth-last-child(an-plus-b | even | odd)

En el siguiente ejemplo usaremos nth-child(3n) que coloreará los elementos 3,6,9... contando desde el final de color azul azul

:nth-of-type()

Selecciona uno o más elementos de un tipo dado, en función de su posición entre un grupo de hermanos. Se especifica con un solo argumento, que representa el patrón para los elementos coincidentes. Sus valores son los mismos que nth-of-type

Sintaxis: :nth-of-child(an-plus-b | even | odd)

En el siguiente ejemplo usaremos p:nth-of-type(1) para poner en negrita el primer párrafo, p:nth-of-type(even) para coloreaer en azul los párrafos pares y p:nth-of-type(odd) para colorear de rojo los párrafo impares. Solo tendrá en cuenta los párrafos, por lo que los divs que hay no contará para nth-of-type

:only-child

Representa un elemento sin hermanos. Esto es lo mismo que :first-child:last-child o :nth-child(1):nth-last-child(1), pero con una especificidad menor.

Sintaxis: :only-child. Hay que destacar que entre la etiqueta y el only-child debe de haber un espacio, es decir, div :only-child ya que div:only-child busca el div que sea hijo único mientras que div :only-child busca el hijo único dentro del div.
